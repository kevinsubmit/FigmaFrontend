# 开发更新 2026-03-20 Backend 注册500与真实业务链路回归

## 本次修复

### 修复 `/api/v1/auth/register` 返回 500

- 现象：真实回归时，注册接口返回 `500 Internal Server Error`
- 根因：`auth.register` 内部在推荐码分支里再次 `from app.crud import coupons as crud_coupons`，把函数顶部已导入的 `crud_coupons` 变成局部变量
- 结果：在更早执行的 `crud_coupons.claim_phone_pending_grants(...)` 位置触发 `UnboundLocalError`
- 处理：删除局部重复导入，统一复用模块顶部导入

文件：

- `backend/app/api/v1/endpoints/auth.py`
