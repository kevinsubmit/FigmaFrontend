# 三端每日签到积分功能

更新时间：2026-04-05

## 目标

- 为已登录用户增加每日签到加积分能力
- 以后端判定为准，避免前端本地时间作弊
- 三端 `My Points` 页面统一展示签到状态和签到入口

## 后端实现

- 新增表：`daily_checkins`
  - 字段：`user_id`、`checkin_date`、`points_awarded`、`created_at`
  - 唯一约束：`(user_id, checkin_date)`
- 新增接口：
  - `GET /api/v1/points/daily-checkin/status`
  - `POST /api/v1/points/daily-checkin`
- 新增配置：
  - `DAILY_CHECKIN_REWARD_POINTS=5`
  - `DAILY_CHECKIN_TIMEZONE=America/New_York`
- 成功签到会：
  - 增加 `available_points`
  - 增加 `total_points`
  - 写入一条 `point_transactions`，`reason = Daily check-in`
- 重复签到保持幂等，不重复加分

## 三端实现

- iOS：`PointsView` 新增签到卡片和签到按钮，成功后刷新积分页与 Profile 摘要

## 当前边界

- `v1` 固定每天签到一次，固定奖励积分
- 暂不支持连续签到、补签、签到通知
- 签到入口先放在三端 `My Points` 页面，不额外扩散到首页
