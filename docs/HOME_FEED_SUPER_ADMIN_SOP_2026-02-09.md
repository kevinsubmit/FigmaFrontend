# 首页图片流超管操作 SOP（中文）

适用对象：仅超管账号  
适用页面：后台 `Home Feed` 管理页（`/admin/home-feed`）

## 一、每日例行操作（建议 5-10 分钟）
1. 打开后台 `Home Feed` 页面，先点一次 `Refresh`。
2. 检查“首页标签顺序预览”是否符合当天运营计划。
3. 检查图片状态：
- 需要上线的图片设置为 `Publish`
- 过期或不展示的图片设置为 `Offline`
- 草稿保留 `Draft`
4. 在 H5 首页抽查：
- 标签展示是否正确
- 点击标签后图片过滤是否正确
- 搜索是否能按标题返回结果

## 二、分类（标签）管理流程
1. 新增分类
- 在 `Create Category` 输入 `Category name`
- 设置 `Sort order`（越小越靠前）
- 勾选 `Active`
- 勾选 `首页标签显示`（若希望 H5 顶部显示）
- 点击 `Create Category`

2. 编辑分类
- 在分类列表点击 `Edit`
- 修改名称、排序、是否首页显示
- 点击 `Update Category`

3. 控制首页是否展示该标签
- `首页标签显示` 勾选：会进入 H5 顶部标签（前提 `Active=true`）
- `首页标签显示` 取消：不在 H5 顶部展示

4. 调整标签顺序
- 在分类列表使用 `↑` / `↓`
- 调整后检查“首页标签顺序预览”是否正确

## 三、图片管理流程
1. 上传并创建图片
- 在 `Create Image` 填写：`Title`、`Description`（可选）
- 上传图片或粘贴 `Image URL`
- 设置状态（建议新图先 `Draft`）
- 勾选一个或多个分类（支持一图多分类）
- 点击 `Create Image`

2. 图片发布
- 在图片列表点击 `Publish`
- H5 端将可见（受分类过滤与专题模式影响）

3. 图片下线
- 点击 `Offline`，H5 不再展示

4. 图片删除
- 点击 `Delete`（软删除）
- 删除后若 H5 暂未刷新，等待缓存刷新或手动刷新页面

## 四、专题模式（节日模式）操作
适用场景：圣诞节、情人节等活动期只展示某类图片。

1. 启用专题
- 在“首页专题模式”勾选 `启用专题模式`
- 选择专题分类（例如 `Christmas`）
- 设置 `start_at` / `end_at`
- 点击 `保存专题设置`

2. 生效判断（需同时满足）
- 已启用专题
- 选择的分类为 active
- 当前时间在起止时间内

3. 关闭专题
- 取消勾选 `启用专题模式` 后保存

## 五、推荐运营规范
1. 标签数量控制
- H5 顶部除 `All` 外，建议额外展示 5-8 个标签，避免过长。

2. 图片排序建议
- 重要图片设置更小 `sort_order`（如 10, 20, 30）
- 预留间隔便于后续插入（不要连续 1,2,3）

3. 发布节奏
- 新图先 `Draft` 校验，再 `Publish`
- 活动结束后统一 `Offline` 活动图

## 六、常见问题排查
1. 问：我把标签设为“首页隐藏”，H5 还显示？
- 先确认该分类 `show_on_home=false` 且保存成功
- 再刷新 H5 页面（必要时清缓存）
- 若仍异常，检查是否为专题模式强制过滤导致认知偏差

2. 问：删除图片后 H5 看到空图位？
- 一般是客户端缓存旧数据，刷新后应恢复
- 若持续存在，检查图片是否确实软删除成功（后台列表是否消失）

3. 问：为什么设置了专题但没生效？
- 检查是否启用
- 检查分类是否 active
- 检查时间窗口是否覆盖当前时间

4. 问：搜索为什么搜不到？
- 当前按图片标题匹配（大小写不敏感）
- 仅返回已发布且未删除数据

## 七、变更记录建议（运营留档）
每次大改后建议记录：
1. 操作人
2. 操作时间
3. 新增/下线图片数量
4. 标签展示顺序调整内容
5. 是否启用专题及时间范围
