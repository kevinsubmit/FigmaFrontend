
  # NailsDash react

  This is a code bundle for NailsDash react. The original project is available at https://www.figma.com/design/9G3L02z2xAsaka8Y3axBcz/NailsDash-react.

## Running the code

Run `npm i` to install the dependencies.

Copy `.env.example` to `.env` and adjust API base URL if needed.

Run `npm run dev` to start the development server.

## Notable routes

- `/settings`
- `/change-password`
- `/phone-management`
- `/language`
- `/feedback-support`
- `/partnership`
- `/privacy-safety`
- `/about`
